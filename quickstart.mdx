## Overview

This guide walks you through making your first API calls with the MT5 Manager API SDK. You'll learn how to authenticate with the MT5 server and retrieve account information.

<Note>
Before starting, make sure you have [installed the SDK](/installation) and have access to an MT5 server with manager credentials.
</Note>

## Prerequisites

To follow this guide, you'll need:

- MT5 server IP address and port (e.g., `127.0.0.1:443`)
- Manager login credentials
- Manager password
- A user login ID to query (for testing)

## Basic workflow

Every MT5 Manager API integration follows this pattern:

<Steps>
  <Step title="Initialize connection">
    Call the `/init/` endpoint to authenticate and receive an access token
  </Step>
  <Step title="Configure SDK">
    Set the access token in the SDK configuration
  </Step>
  <Step title="Make API calls">
    Use the token to make authenticated requests to other endpoints
  </Step>
</Steps>

## Authentication example

The first step is always to initialize the connection and obtain an access token.

### Step 1: Initialize and authenticate

```php initialize.php
<?php
require_once(__DIR__ . '/vendor/autoload.php');

use D4T\MT5Sdk\Configuration;
use D4T\MT5Sdk\MT5Manager\BasicApi;
use GuzzleHttp\Client;

// Configure the SDK
$config = new Configuration();
$config->setHost('http://your-mt5-api-server.com/v1');

// Create the BasicApi instance
$apiInstance = new BasicApi(
    new Client(),
    $config
);

// MT5 server credentials
$server = '127.0.0.1:443';  // Your MT5 server IP:port
$login = 'your_manager_login';  // Manager login
$password = 'your_manager_password';  // Manager password
$timeout = 30000;  // Optional: timeout in milliseconds

try {
    // Initialize and get access token
    $result = $apiInstance->initGet($server, $login, $password, $timeout);
    
    echo "Authentication successful!\n";
    echo "Token: " . $result->getToken() . "\n";
    echo "Response code: " . $result->getCode() . "\n";
    
    // Store the token for subsequent requests
    $accessToken = $result->getToken();
    
} catch (Exception $e) {
    echo "Authentication failed: " . $e->getMessage() . "\n";
    exit(1);
}
```

<Warning>
Never hardcode credentials in production code. Use environment variables or secure configuration management.
</Warning>

### Step 2: Use the access token

Once you have the access token, configure it for subsequent API calls:

```php authenticated-request.php
<?php
// ... (continuing from previous example)

use D4T\MT5Sdk\MT5Manager\UserApi;

// Configure with the access token
$config->setAccessToken($accessToken);

// Create API instance for user operations
$userApi = new UserApi(
    new Client(),
    $config
);

try {
    // Get user information
    $userLogin = '12345';  // The user login ID to query
    $user = $userApi->userUserLoginGet($userLogin);
    
    echo "\nUser Information:\n";
    echo "Login: " . $user->getLogin() . "\n";
    echo "Name: " . $user->getName() . "\n";
    echo "Email: " . $user->getEmail() . "\n";
    echo "Group: " . $user->getGroup() . "\n";
    echo "Leverage: " . $user->getLeverage() . "\n";
    
} catch (Exception $e) {
    echo "Error retrieving user: " . $e->getMessage() . "\n";
}
```

## Complete example

Here's a complete working example that combines authentication and data retrieval:

<CodeGroup>
```php Complete example
<?php
require_once(__DIR__ . '/vendor/autoload.php');

use D4T\MT5Sdk\Configuration;
use D4T\MT5Sdk\MT5Manager\BasicApi;
use D4T\MT5Sdk\MT5Manager\AccountApi;
use GuzzleHttp\Client;

/**
 * MT5 Manager API Quickstart Example
 * 
 * This example demonstrates:
 * 1. Authenticating with the MT5 server
 * 2. Retrieving account information
 */

// Configuration
$config = new Configuration();
$config->setHost('http://your-mt5-api-server.com/v1');

// MT5 credentials
$server = '127.0.0.1:443';
$managerLogin = 'your_manager_login';
$managerPassword = 'your_manager_password';
$testUserLogin = '12345';  // User account to query

try {
    // Step 1: Authenticate and get access token
    echo "Authenticating with MT5 server...\n";
    
    $basicApi = new BasicApi(new Client(), $config);
    $initResult = $basicApi->initGet($server, $managerLogin, $managerPassword);
    
    if ($initResult->getCode() !== 0) {
        throw new Exception("Authentication failed with code: " . $initResult->getCode());
    }
    
    echo "✓ Authentication successful\n\n";
    
    // Step 2: Configure access token
    $config->setAccessToken($initResult->getToken());
    
    // Step 3: Retrieve account information
    echo "Retrieving account information for login {$testUserLogin}...\n";
    
    $accountApi = new AccountApi(new Client(), $config);
    $account = $accountApi->accountLoginGet($testUserLogin);
    
    // Display account details
    echo "\n=== Account Information ===\n";
    echo "Login: " . $account->getLogin() . "\n";
    echo "Balance: $" . number_format($account->getBalance(), 2) . "\n";
    echo "Credit: $" . number_format($account->getCredit(), 2) . "\n";
    echo "Equity: $" . number_format($account->getEquity(), 2) . "\n";
    echo "Margin: $" . number_format($account->getMargin(), 2) . "\n";
    echo "Free Margin: $" . number_format($account->getMarginFree(), 2) . "\n";
    
    echo "\n✓ Quickstart completed successfully!\n";
    
} catch (Exception $e) {
    echo "\n✗ Error: " . $e->getMessage() . "\n";
    exit(1);
}
```

```php With error handling
<?php
require_once(__DIR__ . '/vendor/autoload.php');

use D4T\MT5Sdk\Configuration;
use D4T\MT5Sdk\MT5Manager\BasicApi;
use D4T\MT5Sdk\MT5Manager\UserApi;
use D4T\MT5Sdk\ApiException;
use GuzzleHttp\Client;

function connectToMT5($server, $login, $password) {
    $config = new Configuration();
    $config->setHost('http://your-mt5-api-server.com/v1');
    
    $basicApi = new BasicApi(new Client(), $config);
    
    try {
        $result = $basicApi->initGet($server, $login, $password);
        $config->setAccessToken($result->getToken());
        return $config;
    } catch (ApiException $e) {
        $errorBody = $e->getResponseBody();
        throw new Exception("MT5 authentication failed: {$errorBody}");
    }
}

function getUserInfo($config, $userLogin) {
    $userApi = new UserApi(new Client(), $config);
    
    try {
        return $userApi->userUserLoginGet($userLogin);
    } catch (ApiException $e) {
        if ($e->getCode() === 404) {
            throw new Exception("User not found: {$userLogin}");
        }
        throw new Exception("Failed to retrieve user: " . $e->getMessage());
    }
}

// Main execution
try {
    // Authenticate
    $config = connectToMT5(
        '127.0.0.1:443',
        'manager_login',
        'manager_password'
    );
    
    // Get user information
    $user = getUserInfo($config, '12345');
    
    echo "User: {$user->getName()} ({$user->getEmail()})\n";
    echo "Group: {$user->getGroup()}\n";
    
} catch (Exception $e) {
    echo "Error: {$e->getMessage()}\n";
    exit(1);
}
```
</CodeGroup>

## Creating a new user

Once authenticated, you can create trading accounts:

```php create-user.php
<?php
use D4T\MT5Sdk\MT5Manager\UserApi;
use D4T\MT5Sdk\Models\User;

// Assuming $config is already configured with access token
$userApi = new UserApi(new Client(), $config);

// Create a new user object
$newUser = new User();
$newUser->setName('John Doe');
$newUser->setEmail('john.doe@example.com');
$newUser->setGroup('demo\\demoforex');
$newUser->setPassword('SecurePass123!');
$newUser->setPasswordInvestor('InvestorPass123!');
$newUser->setLeverage(100);

try {
    $result = $userApi->userAddPost($newUser);
    
    echo "User created successfully!\n";
    echo "New login ID: " . $result->getUser()->getLogin() . "\n";
    
} catch (Exception $e) {
    echo "Failed to create user: " . $e->getMessage() . "\n";
}
```

## Configuration options

The Configuration class provides several options for customizing SDK behavior:

```php
$config = new Configuration();

// Set the API host
$config->setHost('http://your-mt5-api-server.com/v1');

// Set access token (after authentication)
$config->setAccessToken($token);

// Enable debug mode
$config->setDebug(true);
$config->setDebugFile('php://output');

// Set custom user agent
$config->setUserAgent('MyApp/1.0');

// Set temp folder for file operations
$config->setTempFolderPath('/tmp');
```

## Common issues

<AccordionGroup>
  <Accordion title="Authentication failed">
    Verify your manager credentials and ensure the MT5 server is accessible. Check that the server address includes both IP and port (e.g., `127.0.0.1:443`).
  </Accordion>
  
  <Accordion title="Connection timeout">
    Increase the timeout parameter in the `initGet()` call. The default is 30 seconds, but slow networks may require more time.
  </Accordion>
  
  <Accordion title="Invalid access token">
    Access tokens may expire. If you receive 401 errors, re-authenticate by calling `initGet()` again to obtain a fresh token.
  </Accordion>
  
  <Accordion title="User not found">
    Ensure the user login ID exists in the MT5 database and that your manager account has permission to access it.
  </Accordion>
</AccordionGroup>
